# CommandX · Sala de Mando — Demo

Demo interactivo de sala de mando de campaña (Sonora) con IA: 7 suites, 89 agentes,
mapas animados, escucha activa multired y flujo multiagente de defensa del voto.

Sitio estático de un solo archivo (`index.html`). Desplegado en commandx.io (Hostinger).

## Despliegue
Hostinger sirve `index.html` en la raíz (`public_html`). Actualizar = push a `main`
y desplegar desde hPanel → Avanzado → GIT.
